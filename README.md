# 🛒 Customer Behavior Analysis (Excel • SQL • Power BI • Python)

---

## 📊 Project Overview
This project is a deep analysis of retail/e-commerce customer transaction data to understand purchasing behavior, revenue patterns, discount effectiveness, and customer segmentation.

The goal is to convert raw transactional data into actionable business insights that help improve marketing strategies, customer targeting, and overall business performance.

---

## 📁 Dataset
Retail / E-commerce transactional dataset containing customer purchase details such as:
- Age
- Category
- Purchase Amount
- Payment Method
- Season
- Previous Purchases

---

## 🛠️ Tech Stack & Tools
- Data Analysis: Python (Pandas, NumPy)  
- Data Cleaning & EDA: Matplotlib, Seaborn  
- Database Queries: SQL (PostgreSQL)  
- Visualization: Power BI  
- Process: Data Cleaning, EDA, Feature Analysis, Business Reporting  

---

## 🧱 Project Workflow

### 1. Data Cleaning & Preprocessing (Python)
- Handled missing values and duplicates  
- Standardized categorical values  
- Checked data consistency and structure  
- Prepared dataset for analysis and visualization  

---

### 2. Exploratory Data Analysis (EDA)
- Analyzed customer purchase patterns  
- Studied category-wise and seasonal trends  
- Identified high-value customer segments  
- Explored relationship between discounts and purchase behavior  

---

### 3. SQL Analysis
- Performed aggregation and filtering queries  
- Extracted insights on revenue, customers, and categories  
- Analyzed repeat purchases and customer segmentation  

---

### 4. Power BI Dashboard
- Built interactive dashboards for business insights  
- Visualized revenue distribution, customer segments, and trends  
- Created KPI cards for key business metrics  

---

## 📌 Key Insights

- Electronics category contributes nearly half of total revenue  
- Subscribers spend significantly more than non-subscribers  
- Customers aged 60+ generate a large portion of revenue  
- Express shipping users tend to have higher order values  
- Seasonal trends show peak revenue during spring  

---

## 📈 Business Impact

- Helps identify high-value customer segments  
- Improves marketing targeting strategies  
- Supports pricing and discount optimization  
- Enhances customer retention strategies  
- Provides data-driven decision-making insights  

---

## 🎯 Conclusion
This project demonstrates how raw e-commerce transaction data can be transformed into meaningful business insights using Python, SQL, and Power BI.

It highlights customer behavior patterns that can help businesses improve revenue, customer engagement, and overall strategy.
